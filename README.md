# Overview

Your PT that you train!
LiftBro is an electron application that allows you to teach a ML model what several excerises look like by demonstration, and then have LiftBro track your workout by time spent on different movements!

## Built With
* Node.js
* Yarn
* React.js
* React Router DOM
* React Webcam
* Electron
* Material-UI
* TensorFlow
* PoseNet

# Getting Started

To get started:
1. Install the latest npm
### `npm install npm@latest -g`
2. Clone the repo
### `git clone https://github.com/Danryanh7/liftbro.git`
3. Install dependancies
### `npm i`
4. Serve using electron
### `npm run electron:serve`
